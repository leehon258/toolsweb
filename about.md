# 关于本站

## 我们是谁

**小众工具集** 是一个专注于「独立开发者精品工具」的测评与对接平台。

我们不追大厂热门，也不做无脑搬运。我们只做一件事：**把那些被埋没、却真正好用的独立工具，挖出来、测明白、送到需要它的人面前。**

## 我们坚持什么

- **真实客观**：每一篇测评都来自亲测实测，优缺点并陈，不收钱写软文。
- **深度拆解**：不仅告诉你「好不好用」，还拆解「为什么好用 / 差在哪 / 适合谁」。
- **尊重开发者**：独立开发者不易，我们尽力让好作品被看见、被认可。

## 平台能为你做什么

| 角色 | 你能获得 |
| --- | --- |
| 普通用户 | 高质量工具推荐、真实测评避坑、按场景找工具 |
| 独立开发者 | 作品曝光、用户反馈沉淀、需求对接与共创机会 |
| 内容读者 | 水下宝藏专栏、编辑精选、月度 / 年度盘点 |

## 联系与合作

- 商务 / 投稿 / 纠错：欢迎在任意工具页底部评论区留言
- 仓库与源码：见页脚 GitHub 链接

> 本站为纯静态站点，内容由编辑团队持续更新。部分内容可能由 AI 辅助生成，最终测评结论均为人工审核。
